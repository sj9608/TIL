# jQuery

JS 를 이용해 만든 다양한 함수들을 모아놓은 오픈소스 라이브러리로 모든 웹 브라우저에서 동작한다.

[jQuery에 대한 정보](https://learn.jquery.com) , [jQuery API 문서](https://api.jquery.com)



## jQuery 특징

* 웹 표준 및 브라우저간 호환성이 좋음
* Ajax 호출 방법이 편리함

## jQuery 사용 이유

* DOM 조작이 쉬워진다.
* 동적인 효과를 구현하기 쉽다.
* Ajax도 마찬가지로 쉽게 구현할 수 있다.

## jQuery 사용 방법

* 기본적으로 CDN 방식을 이용한다 
* $('**선택자**').메소드 형태를 지닌다.
